Constructor.

If the language or environment is None, the glossary default is used.

References in the text are linked as base#ckey, where ckey is the concept key as pointed to by the reference, and base is the page where the concept is anchored. The refbase parameter is the mapping of concept keys to pages. If the mapping is not given, base will always be empty; if the mapping is given and the key that a reference points to is not present in it, no link will be made.

Parameters:
  • gloss (Glossary) - the glossary to which the text belongs
  • lang (string or None) - the language for which the text is formatted
  • env (string or None) - the environment for which the text is formatted
  • prefix (string or None) - prefix to add to the text
  • suffix (string or None) - suffix to add to the text
  • refbase (dict of string:string) - mapping of concept keys to source pages
  • wtag (string or None) - tag to wrap the resulting text with
  • wattrs (dict or None) - attributes to the wrapping tag, as name->value mapping
  • wcond (bool) - add wrapping tag only if not wrapped with it as it is
  • pclass (string or None) - class attribute to assign to paragraphs

(Call operator)

 

Format the text.

Prefix/suffix and wrapping given by the constructor may be overriden here. This is useful e.g. for lists.

The formatted text is stripped of leading and trailing whitespace.

Parameters:
  • text (instance of Text) - the text to be formatted
  • prefix (string or None) - overriding prefix for the text
  • suffix (string or None) - overriding suffix for the text
  • wtag (string or None) - tag to wrap the resulting text with
  • wattrs (dict or None) - attributes to the wrapping tag, as name->value mapping
  • wcond (bool or None) - add wrapping tag only if not wrapped with it as it is
  • pclass (string or None) - class attribute to assign to paragraphs
Returns: string
formatted HTML text
