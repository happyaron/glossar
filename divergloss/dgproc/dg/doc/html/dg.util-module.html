Function Details

langsort(lst, lang=None)

 

Sort a list of words using collation of the given language.

If lang is None, current locale is used for collation.

Parameters:
  • lst (list) - list to sort, in-place
  • lang (string of None) - language for the collation

langsort_tuples(lst, index, lang=None)

 

Sort a list of tuples by comparing elements of given index, using collation for given language.

If lang is None, current locale is used for collation.

Parameters:
  • lst (list of tuples) - list to sort, in-place
  • index (int) - sort by tuple elements at this position
  • lang (string of None) - language for the collation
