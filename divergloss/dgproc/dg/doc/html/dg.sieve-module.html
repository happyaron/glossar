Glossary sieves.

Submodules
  • dg.sieve.html: Create HTML pages out of the glossary.
  • dg.sieve.html_bidict: Create HTML table with bilingual dictionary.
  • dg.sieve.plrules: Update terminology rules file for Pology's check-rules sieve.
  • dg.sieve.po: Create a PO file out of the glossary.
  • dg.sieve.sr_latin: Transform Serbian Cyrillic text in the glossary into Serbian Latin.
  • dg.sieve.tbx: Create a TBX view of the glossary.
  • dg.sieve.text_simple: Create a simple plain text view of the glossary.
