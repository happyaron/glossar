child_nodes(gnode, gtype=None)

 

Get child nodes of a glossary node in a flat list.

Child nodes may be the attributes of the given node by themselves, or stored in list, dict, or Dset attributes of the given Gnode.

If a particular node type (subclass of Gnode) is given, than only the nodes with this type will be selected. If several types are wanted, they can be given as a sequence.

Parameters:
  • gnode (Gnode) - a glossary node
  • gtype (subclass of Gnode, or a sequence thereof) - a type of glossary nodes
Returns: list of Gnode
